# Prompt

---

안녕. 네가 개발해준 앱 덕분에 삼국지 14 천하통일을 눈앞에 두고 있어.

마지막 내정 작업 전, 추가로 꽤 대규모 수정을 요청하려고 해.

## 수정계획

- 나는 코드를 잘 모르지만, 현재 저장소의 코드구조는 거의 추상화되어있지 않은 것 같아.
- 이에 따라 2단계에 걸쳐서 추상화를 진행하려고 해.
    - 1단계 : 저장소 내부 스크립트들을 역할별로 구분. (객체지향 느낌)
    - 2단계 : 하드코딩되어 있는 조건들의 추상화. (예를 들어, 모병요원으로 어떤 수치조건을 요구하는지를 내가 앱에서 직접 기입하고 싶어.)

## 요청

- 우선, 저장소 전체의 데이터들과 코드베이스를 읽어서 1단계를 위한 계획만을 작성해줘.

---

OK! 고생했어. 2단계까지 완료 후에, 직접 게임플레이에 사용해보면서 테스트해볼 예정이야. 바로 2단계 작업계획을 작성해줘.

- 2단계 : 하드코딩되어 있는 조건들의 추상화. (예를 들어, 모병요원으로 어떤 수치조건을 요구하는지를 내가 앱에서 직접 기입하고 싶어.)

---

좋아. 2가지만 더 추가로 요청할게. 계획에 반영해줘.

## 요청

1. 현재 내정요원 배치 후, 남는 무장의 목록을 "내정관리" 가장 아래에 목록으로 배치하고 있는 것으로 알고 있어. 이 목록에 통/무/지/정/매를 모두 표시해주고, 1줄당 1명만 나오게끔 수정해줘. (보유무장 탭과 유사한 목록이지만, 개성은 생략해줘.)

2. 교역요원 조건을 하나 더 추가해서 내가 선택할 수 있게끔 해줘.
    - 현재 : 교역 잔여 정렬기준은 |ip - 100|인데 이 방식은 선택지로 유지.
    - 추가 : 각 도시에 배정되지 않은 요원 중 '정치'스탯이 가장 높은 무장들을 우선 배정하는 조건을 선택지로 추가.
    - 유지 : '특사' 개성에 따른 교역요원 우선배정 로직은 유지.
